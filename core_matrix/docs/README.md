# CoreMatrix Product Docs

This directory is the product-facing documentation entrypoint for CoreMatrix.

Use these documents first:

- [INSTALL.md](/Users/jasl/Workspaces/Ruby/cybros/core_matrix/docs/INSTALL.md): deploy CoreMatrix on a Linux host with Docker Compose
- [ADMIN-QUICK-START-GUIDE.md](/Users/jasl/Workspaces/Ruby/cybros/core_matrix/docs/ADMIN-QUICK-START-GUIDE.md): bootstrap an installation and perform first operator tasks
- [INTEGRATIONS.md](/Users/jasl/Workspaces/Ruby/cybros/core_matrix/docs/INTEGRATIONS.md): provider and IM integration guidance

## Scope

These docs describe how to use and operate CoreMatrix as a product.

They are intentionally different from the monorepo's historical design and
implementation records. Architecture plans, behavior notes, and retired
execution documents were archived for traceability and should not be treated as
the primary operator-facing truth.
