# Proposed Plans Index

Use this directory for early plan drafts that are not ready to become active
execution plans.

Place material here when:

- a future phase has rough milestones or work slices
- a plan still depends on unresolved design questions
- the work should not enter `docs/future-plans` or `docs/plans` yet

Move a document out of this directory when:

- the plan is accepted but intentionally deferred into `docs/future-plans`
- the plan becomes the current execution queue and belongs in `docs/plans`
