# Finished Plans Index

This directory is reserved for completed execution plans, milestones, and related records that have passed their stage-level acceptance checks.

Use it only after active work has been completed and verified. Do not place
in-progress execution documents here.

Finished records should be historically self-contained. If a completed plan,
milestone, or review used `references/` or external repositories during
implementation, summarize the retained findings in the finished record instead
of leaving only a volatile reference path behind.
