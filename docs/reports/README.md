# Reports

This directory now stores authored reports that are part of the repository,
such as architecture audits and narrative write-ups.

Reports are preserved snapshots of what was true at the time they were written.
Earlier reports may therefore retain superseded naming and pre-reset
assumptions. Treat them as historical diagnosis, not as the current product
contract.

Generated verification logs, proof bundles, screenshots, exports, and other
runtime artifacts no longer belong here. The top-level verification project owns
those outputs and writes them to:

- `/Users/jasl/Workspaces/Ruby/cybros/verification/logs/`
- `/Users/jasl/Workspaces/Ruby/cybros/verification/artifacts/`

Those runtime outputs are intentionally gitignored.
