# Core Matrix Phase 7 Extension And Plugin Follow-Up

## Status

Deferred future plan.

## Purpose

Consider extension and plugin packaging only after the kernel and multiple
agent shapes are already stable.

## Preconditions

- the loop is proven
- the Web product is usable
- additional agents have validated that the kernel is not Fenix-shaped
- trigger and channel work have settled

## Design Rule

Any future plugin system must register into the same capability-governance
model instead of creating a parallel execution architecture.
