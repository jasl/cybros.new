# Core Matrix Phase 6 Client And Channel Follow-Up

## Status

Deferred future plan.

## Purpose

Extend Core Matrix beyond the initial Web product surface without changing the
kernel's loop authority.

## Scope

- IM integrations
- PWA
- desktop clients
- channel-specific delivery and interaction affordances
- agent-defined composer completions mounted into additional clients

## Design Rule

Client and channel work may add new interaction surfaces, but it must reuse the
same kernel loop semantics, feature policy, and capability governance.
